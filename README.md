# simple-hello-world-example
A Simple Hello World Example Package

## Installation
Install via NPM:

```bash
npm install npm-yousef

```

## Usage

#### javascript

```javascript

var myApp = require("npm-yousef");

myApp.printMsg(); // print and return " Hello World! " 

```

#### TypeScript

```typescript

import * as myApp from 'npm-yousef';

myApp.printMsg(); // print and return " Hello World! " 


```
